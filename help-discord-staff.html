gavelBan Player Via Discord

  • Ban syntax:/player-level <factorioName> <level> <ban reason>
  • Example quick-ban/player-level grieferName banned logged in and started deleting everything: https://m45sci.xyz/u/fact2/new-logs/a/game-01-April-2069.log
  • Player Levelsbanned, new, member, regular, veteran, moderator
    • Note: We do not casually ban & unban. Please use banish / jail for time-outs if you are not certain.

uploadUpload maps

You can attach one of each type at the same time.

  • Upload a save-game/upload save-game <file attachment>
  • Replace mod-list file/upload mod-list <file attachment>
  • Replace mod-settings file/upload mod-settings <file attachment>
    • Mod settings files do not contain map-specific settings, only global settings.

extensionChange installed mods

  • Show installed mods/editmods list-mods
  • Enable mods by name/editmods enable-mod <mod1,mod2>
  • Disable mods by name/editmods disable-mod <mod1,mod2>
  • Add mods/editmods add-mod <mod-url-1,mod-name-2>
    • Add-mod supports mod names OR mod-portal URLs
  • Erase all mod files/editmods clear-all-mods
  • Show mod add/remove/update history/editmods mod-history

autorenewMap Reset Scheduling

  • Reset interval/map-schedule interval-months <number> interval-weeks <number> interval-days <number> interval-hours <number>
Use one: 'interval-weeks 2' or combine intervals: 4 months, 3 weeks, 2 days and 1 hour
  • Manually set next reset date & time/map-schedule reset-date <YYYY-MM-DD HH:MM:SS>
  • Don't automatically schedule next map reset/map-schedule disable

mapChange Map

  • Rewind map menu/change-map <press enter>
  • See older maps/change-map list
  • Load older map/change-map load <save name>
    • Unless the save you are after is more than 25 autosaves ago, just press enter to get a vote-map like menu.
  • Archive, then generate new/map-reset
  • Generate a new map/factorio new-map
  • Send to map archive/factorio archive-map
  • Install mods found in save-game/factorio sync-mods
  • Update all mods/factorio update-mods

tuneMap Settings

  • Change map name/config-server name <name>
  • Member or higher only/config-server members-only <true|false>
  • Regular or higher only/config-server regulars-only <true|false>
  • Map-preset/config-server map-preset default, rich-resources, marathon, death-world, death-world-marathon, rail-world, ribbon-world, island, lakes
  • Friendly fire/config-server friendly-fire <true|false>
  • Pause game when empty/config-server auto-pause <true|false>
  • Game speed for current map/config-server speed <value>
  • Minutes between autosaves/config-server autosave-mins <min>
  • AFK idle kick minutes/config-server afk-kick-mins <min>
  • Seed for next new map/config-server map-seed <seed>
