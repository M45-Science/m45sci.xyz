badgeMembership & Roles

  • Auto‑Register: Roles assigned by in-game score (non-idle); no manual signup.
  • Member: Earn by playing public maps; needed for private-server access.
  • Regular: Access private servers & channels; extra vote power for banish.
  • Veteran: Two point votes; solo reset/rewind; can banish Regulars.

  • In-game help: click the top-left icon 'help-info'

chatDiscord Commands

Commands work only in a map-specific Discord channel. (see Map Channel Status below)

  • warning/register — Run this first and follow instructions.
  • how_to_vote/vote-map — Rewind, reset, or skip scheduled map reset.
  • info/info [verbose] — Show server details (add verbose for extra stats).
  • download/modpack — Get a link to download installed mods.
  • group/players — List current players online.
  • pause_circle/pause-game — Pause the map (2 min timeout) while you connect.
  • search/whois search:<player> — Lookup player info.
  • scoreboard/scoreboard — Show top 40 players.
  • extension/list-mods — Show mods on current map.
  • label/get-roles — Opt in/out Notification Roles.

See the Official Discord guide for app usage.

chat_bubbleMap Channel Status

Channels like ⚫a-planetoid-monthly bridge Discord↔Factorio. Status icons:

  • ⚪Public map — anyone can join
  • 🟢Members-only — @Member required
  • 🟠Regulars-only — @Regular required
  • 🔴Veterans/invite-only — @Veteran+ required
  • ⚫No players online

If prefixed by a number, that indicates current player count. All Discord commands listed above work here.

restoreInactive Accounts

  • Haven’t played recently? Join any public server to “thaw” your account.
  • After a few minutes, members-only servers update your status.
  • Whitelist limited to the 1,500 most recently active members/regulars (of 5,000+).
  • Veterans & Moderators always remain whitelisted.

receipt_longLogs & Activity Viewer

  • 💾Autosaves: Filename where the game saved; use these to rewind to specific points.
  • Messages start with FactorioTime (days:hours:minutes:seconds) and local real-world time.
  • To load an old save (more than 25 autosaves back), ask in the ❓moderation-help channel.
  • Correlate Factorio time with production graphs to rewind before damage. Note: game pauses when empty.
  • Browse all logs: Activity Logs. Sort by “Last modified” for date order.
  • Factorio log files named game-YYYY-MM-DD.log. Live viewer: Live Log Viewer.
  • Logs include UTC timestamps. Convert to your zone: Timezone Converter.
  • Sample entry: 18:21:33: [JOIN] NewPlayer joined the game.
  • GPS tags like [gps=-858.5,-911.5] can be pasted in-game to pinpoint locations.

This data is invaluable for identifying griefers with no speculation.
